Features

  • Written in Java.
  • Based on Airspace Frenzy 6.
  • Uses Maven for automated builds, running tests and dependency management.
  • Low resource usage, approximately 135mb of RAM required.
  • Nicely formatted, readable, well commented code.

Project Setup (For Eclipse)

  1. Clone the project from the git repository:
    git clone https://github.com/SEPR13-SAW/atc-game.git
  2. Set up the project (done automatically with the Maven Eclipse plugin) or manually using:
    mvn eclipse:eclipse
  3. Final builds can be done using:
    mvn assembly:assembly
