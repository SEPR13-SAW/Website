Plane Panic

Source Code Documentation Manual Play

Features

  • Written in Java.
  • Uses Lombok to avoid having to manually write getters, setters and some other common functions.
  • Uses Maven for autmoated builds and dependency management.
  • Low resource usage, approximately 135mb of RAM required.
  • Nicely formatted, readable, well commented code.

Project Setup (For Eclipse)

  1. Install Lombok for Eclipse.
  2. Clone the project from the git repository:
    git clone https://github.com/SEPR13-SAW/Plane-Panic.git
  3. Set up the project (done automatically with the Maven Eclipse plugin) or manually using:
    mvn eclipse:eclipse
  4. Final builds can be done using:
    mvn assembly:assembly
